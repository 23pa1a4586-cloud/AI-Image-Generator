# AI Image Generator

An AI-powered web application that transforms text prompts into high-quality images using an AI image generation service. The application provides a clean, responsive interface where users can generate, preview, and download images with ease.

## Features

- Generate images from text prompts
- Responsive and modern user interface
- Image preview
- Download generated images
- Loading indicator during generation
- Error handling for invalid requests
- Backend integration using Supabase Edge Functions

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- AI Image Generation API


## Future Improvements

- Support multiple image styles
- Image generation history
- User authentication
- Favorite images
- Enhanced prompt suggestions
